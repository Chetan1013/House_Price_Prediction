# 🏠 House Price Prediction Using Machine Learning

A Flask-based Machine Learning web application that predicts house prices based on:
- Location
- BHK
- Bathrooms
- Square Feet Area

The project uses:
- Python
- Flask
- Scikit-learn
- Pandas
- Bootstrap

---

# 🚀 Features

✅ Predict house prices instantly  
✅ Clean and responsive UI using Bootstrap  
✅ Machine Learning model integration  
✅ Flask backend support  
✅ Bangalore house dataset support
